GitHub 文件加速站

快速下载GitHub上的资源文件

使用提示

GitHub文件链接带不带协议头都可以，支持release、archive以及文件。

release、archive使用cf加速，文件会跳转至JsDelivr。

注意，不支持项目文件夹。

示例链接
  • 分支源码：https://github.com/zhuangzhi213/project/archive/main.zip
  • release源码：https://github.com/zhuangzhi213/project/archive/v1.0.0.tar.gz
  • release文件：https://github.com/zhuangzhi213/project/releases/download/v0.1.0/example.zip
  • 分支文件：https://github.com/zhuangzhi213/project/blob/master/filename
